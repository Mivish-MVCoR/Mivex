# Mivex

A secure, lightweight and flexible chest shop plugin for Paper servers.

Mivex allows players to create protected chest shops using simple commands and a unique key system. The plugin focuses on security, performance and ease of use while remaining highly configurable.

---

## ✨ Features

- 🔑 Protected chest shops
- 👻 Phantom Membrane economy
- 📦 Buy & Sell shops
- 💾 Automatic shop backups
- 🛡 Lost shop recovery system
- 📋 Personal shop management
- ⚡ Fast command-based interface
- 🎯 Smart Tab Completion
- 🔒 Secure shop keys
- 🔧 Administrator management tools

---

## 📋 Commands

| Command | Description |
|---------|-------------|
| `/cshop` | Create a new shop |
| `/cshop edit` | Edit an existing shop |
| `/cshop mylist` | View your shops |
| `/cshop list` | View all shops (Admin) |
| `/cshop list <player>` | View a player's shops (Admin) |

---

## 📦 Requirements

- Java 17+
- Paper 1.19.4+

---

## 🚀 Installation

1. Download the latest release.
2. Place **Mivex.jar** into the `plugins` folder.
3. Restart your server.
4. Configure the plugin if needed.
5. Enjoy!

## Про плагін

Mivex — це безпечний, легкий та гнучкий плагін магазинів для серверів Paper.
